Akut-p-piller

Akut-p-piller är en tablett som en tjej kan ta efter ett oskyddat samlag för att minska risken att bli gravid. Ju tidigare man tar tabletten desto mindre risk är det att man blir gravid. Man kan använda akut-p-piller om man glömt att skydda sig eller det skydd man använde inte funkade.

Akut-p-piller är ett sätt att försöka förhindra graviditet om man av någon anledning haft sex utan att använda skydd.

Man bör också ta ett akut-p-piller om det finns en risk för att det skydd man använde inte funkade, till exempel om kondomen sprack eller om man kräkts upp sitt p-piller.

Tabletten som man tar innehåller ett hormon som gör att man inte får någon ägglossning eller att ägglossningen skjuts fram. Akut-p-piller avbryter alltså inte en graviditet och är därför inte samma sak som en abort eller ett så kallat abortpiller.

Akut-p-piller ska bara användas som en nödlösning och inte som ett regelbundet skydd mot graviditet.

Hur använder man akut-p-piller?

Ibland kallas akut-p-piller för dagen-efter-piller. Det är lite missvisande eftersom man inte måste ta tabletten dagen efter samlaget. Men ju tidigare man tar tabletten desto mindre risk är det att man blir gravid. 

Det finns två sorters akut-p-piller. En sort som man kan ta upp till tre dygn efter ett samlag. Det finns också ett akut p-piller som man kan ta upp till fem dygn efter samlag. Den sorten kan man bara få på en ungdomsmottagning eller på recept av barnmorska eller läkare. Om man behöver ett sånt recept kan man kontakta en ungdomsmottagning, sex- och samlevnadsmottagning, vårdcentral, barnmorskemottagning eller kvinnoklinik. Oftast betalar man för det. På en del mottagningar kan man få det utan att det kostar något.

När man tagit ett akut-p-piller är det vanligt att mensen kommer antingen lite tidigare eller senare än väntat. Det är också vanligt med mellanblödningar. Oavsett om man får sin mens som väntat eller inte ska man alltid göra ett graviditetstest efter tre till fyra veckor för att vara säker på att man inte blivit gravid.

Efter att man tagit ett akut-piller kan man må illa, få huvudvärk och ont i magen. Det är inte farligt och brukar gå över efter något dygn.

Var kan man få tag på akut-p-piller?

Man kan få hjälp med akut-p-piller på de flesta ungdomsmottagningar, hud- och könsmottagningar och vårdcentraler. Man kan även få akut-p-piller på en del barnmorskemottagningar och kvinnokliniker. Det finns också en sorts akut p-piller att köpa på ett apotek utan recept.

Om man går till exempelvis en ungdomsmottagning för att få akut-p-piller kan man även få en ny tid för att göra graviditetstest och, om man vill, ett klamydiatest. Man kan också få råd om hur man kan skydda sig mot graviditet och könssjukdomar framöver om man vill ha det.

Vad kostar akut-p-piller?

Om man köper akut-p-piller receptfritt på ett apotek får man betala för det. Hur mycket det kostar kan variera beroende på vilket apotek man går till. På ungdomsmottagningen får man tabletten kostnadsfritt.

Hur säkert är akut p-piller?

Risken att akut-p-pillret inte funkar, det vill säga att man blir gravid, ökar ju längre tid efter samlaget man tar pillret.

Om man kräks inom tre timmar efter att man tagit tabletten eller om man har diarré är det inte säkert att kroppen hinner ta upp hormonet. Därför måste man ta en ny tablett så fort som möjligt, annars finns det risk att man blir gravid.

Ordförklaring

Bipacksedel
Den skriftliga information som följer med i förpackningen till alla läkemedel. Bipacksedeln finns även att läsa på www.fass.se.
Menscykel
Menscykel kallas tiden från första dagen på en mens till första dagen på nästa. Cykel är ett annat ord för ring och betyder att någonting är återkommande.

Vad som gäller för just det akut-p-piller man tar kan man läsa i bipacksedeln. Bipacksedeln är den lapp som ligger i förpackningen när man får eller köper sitt akut-p-piller. Man kan alltid fråga en barnmorska på till exempel en ungdomsmottagning eller hud- och könsmottagning om man känner sig osäker.

För att vara säker på att man inte blivit gravid bör man göra ett graviditetstest ungefär tre veckor efter att man tagit tabletten.

Hur ofta kan man ta akut-p-piller?

Det finns ingen gräns för hur många gånger per menscykel man kan ta ett akut-p-piller. Har man haft oskyddat samlag och inte vill bli gravid är det därför bättre att ta ett akut-p-piller flera gånger under samma menscykel än att låta bli, även om risken för mellanblödningar ökar.

Behöver man ta akut-p-piller ofta bör man tänka över hur man skyddar sig vid samlag och om man kan behöva ändra på det. För att få hjälp med det kan man kontakta en ungdomsmottagning, barnmorskemottagning eller hud- och könsmottagning.

Fördelar

  • Bra nödlösning för att minska risken för graviditet om man haft samlag utan att skydda sig.

Nackdelar

  • Akut-p-piller skyddar inte mot könssjukdomar.
  • Hormonet i akut-p-piller kan påverka en så att man tillfälligt får till exempel mellanblödningar, huvudvärk eller illamående.

Anmäla biverkningar av läkemedel

Om man är orolig eller har stora besvär av biverkningar från ett läkemedel ska man i första hand kontakta den läkare eller annan sjukvårdspersonal som skrivit ut läkemedlet. Man kan också själv rapportera biverkningar av ett läkemedel till Läkemedelsverket. Man kan rapportera in misstänkta biverkningar från alla läkemedel, både receptbelagda och receptfria.

Man behöver inte vara säker på att biverkningen beror just på läkemedlet, det räcker att man tror det. Man kan rapportera direkt på läkemedelsverkets webbsida eller genom att skicka in en rapport.
